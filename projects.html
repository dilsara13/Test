Pizza Hub

PizzaShop is a React Native mobile app with Firebase integration that lets users securely create accounts with their name, email, and password. The signup screen features a clean UI, loading indicators, and real-time data storage in Firestore. Built with a modular structure, it ensures smooth performance and easy scalability for future pizza ordering features.

Tech: React Native (with TypeScript),Firebase Firestore,React Hooks (useState)

User Management System (React MERN)

Modern React & Node.js web application for managing users efficiently. It features a responsive frontend with forms and tables to add, update, and delete users, including details like ID, Name, Age, Contact No, Email, and Position. The Position field uses a dropdown with a clear placeholder and displays color-coded badges for easy visual identification. The backend is powered by Node.js, Express, and MongoDB, ensuring reliable data storage and seamless CRUD operations.

Tech:React,Material-UI,Node.js,Express,MongoDB
